Acerca del conjunto de datos
Este proyecto brinda la oportunidad de analizar y predecir datos de solicitudes de tarjetas de crédito. El conjunto de datos incluye diversos atributos, como ingresos, edad, ocupación e historial crediticio previo, que pueden influir en la decisión de aprobación de la tarjeta.

El objetivo de este proyecto es explorar el conjunto de datos, limpiarlo y generar las características necesarias para construir modelos de aprendizaje automático que puedan predecir la probabilidad de aprobación de una tarjeta de crédito.

Utilizaremos diferentes algoritmos de aprendizaje automático, como regresión logística, árboles de decisión y bosques aleatorios, y los evaluaremos para determinar el modelo de mejor rendimiento.

Nuestro objetivo es desarrollar un modelo que demuestre un buen rendimiento y prediga con precisión las decisiones de aprobación de tarjetas de crédito en situaciones reales. Con este proyecto, buscamos comprender el riesgo crediticio y ayudar a las compañías de tarjetas de crédito en la toma de decisiones.
